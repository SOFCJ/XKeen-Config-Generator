# 🚀 SOFCJ XKeen Config Generator ✗

[![GitHub pages](https://img.shields.io/badge/GitHub%20Pages-Active-success?style=flat-square&logo=github)](https://sofcj.github.io/XKeen-Config-Generator/)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg?style=flat-square)](https://opensource.org/licenses/MIT)

A fast, private, and responsive web-based configuration generator for XKeen/V2Ray **outbounds**. It effortlessly converts standard connection links into clean, structured JSON format.

## ✨ Features

* ⚡ **Local Generation:** All links are processed directly within your browser. Your keys, passwords, and UUIDs are completely safe and never sent to any third-party servers.
* 🎨 **Modern Interface:** Sleek, minimalist glassmorphic UI featuring a smooth animated gradient outline on control buttons.
* 🌓 **Dual Themes:** Full support for both Dark and Light modes, saving your preference automatically via `localStorage`.
* 🛠️ **Smart Warnings:** Built-in validation checks (e.g., advising port 443 usage for enhanced stealth and masquerading).
* 📋 **Convenient Export:** Copy the generated JSON to your clipboard with a single click or download it directly as a correctly named `04_outbounds.json` file.
* 🌐 **Dual language support** (RU/EN) with automatic settings saving.
* 
## 📡 Supported Protocols

* **VLESS** (including Reality / TCP / gRPC / WS)
* **VMess** (with Base64 payload decoding, supporting kcp, ws, h2 network types)
* **Trojan** (with TLS / Reality support)
* **Shadowsocks (SS)**

---

## 🚀 How to Run Locally

If you prefer not to use GitHub Pages, you can easily host and run this project on your own machine:

1. Clone this repository or simply download the `index.html` file.
2. Open `index.html` using any modern web browser.
3. Paste your subscription/connection link and click **Generate Config**.
