# 🚀 XKeen Config Generator

[![GitHub pages](https://img.shields.io/badge/GitHub%20Pages-Active-success?style=flat-square&logo=github)](https://sofcj.github.io/XKeen-Config-Generator/)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg?style=flat-square)](https://opensource.org/licenses/MIT)

Быстрый, конфиденциальный и адаптивный веб-генератор конфигурационных файлов **outbounds** для конфигураций XKeen/V2Ray. Преобразует стандартные ссылки подключения в чистый структурированный JSON.

## ✨ Особенности

* ⚡ **Локальная генерация:** Все ссылки обрабатываются прямо в вашем браузере. Никакие ключи, пароли и UUID не отправляются на сторонние серверы.
* 🎨 **Современный интерфейс:** Минималистичный стеклянный дизайн (Glassmorphic UI) с плавной анимацией бегущего контура кнопок.
* 🌓 **Две темы:** Полноценная поддержка тёмной и светлой тем с сохранением вашего выбора в `localStorage`.
* 🛠️ **Умные предупреждения:** Автоматическая проверка портов (например, рекомендация использовать порт 443 для повышенной маскировки).
* 📋 **Удобный экспорт:** Копирование JSON в один клик или сохранение готового файла под правильным именем `04_outbounds.json`.

## 📡 Поддерживаемые протоколы

* **VLESS** (включая Reality / TCP / gRPC / WS)
* **VMess** (с поддержкой расшифровки Base64, типов сети kcp, ws, h2)
* **Trojan** (с поддержкой TLS/Reality)
* **Shadowsocks (SS)**

## 🚀 Как использовать локально

Если вы не хотите использовать GitHub Pages, вы можете запустить проект у себя на компьютере:

1. Склонируйте репозиторий или скачайте файл `index.html`.
2. Откройте `index.html` в любом современном браузере.
3. Вставьте вашу подписку/ссылку и нажмите **Сгенерировать конфиг**.
